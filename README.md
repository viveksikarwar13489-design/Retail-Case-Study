# Retail-Case-Study
This project focuses on analyzing retail transaction data to generate meaningful insights about customer behavior, product performance, and business growth.
In today’s competitive retail environment, understanding customers and optimizing business processes using data analytics is very important. This project uses customer, transaction, and product data to uncover patterns such as purchasing behavior, popular product categories, and sales trends.

The goal is to help retailers make data-driven decisions to improve customer satisfaction and increase profitability.

📂 Dataset Information

The dataset is provided in an Excel file: Retail Data.xlsx, which contains 3 sheets:

Customer
Customer demographics (age, gender, city, etc.)
Transactions
Customer purchase and return details
Product Hierarchy
Product categories and sub-categories
